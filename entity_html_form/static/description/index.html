Description

Creates an HTML form that you can embed anywhere

Features

  • Currently works with field types char, text and integer
  • Currenlty outputs textbox, textarea and number html input types

Using the module

Creating the form

1. Go to Settings->Technicial->HTMl Embed Form->Create Forms and select the fields you want in the form
2. Click the "Make Embed Code" button and paste the code in a website

Viewing Form Submittions

1. Go to Settings->Technicial->HTML Embed Form->Inserted Form Data
