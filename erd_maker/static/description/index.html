Description

Creates an ERD starting from the specified table and transending a certain depth

Automatic ERD

Generate an ERD with little to no effort.

Instructions
1. Go to 'Settings->Migration->Create ERD'
2. Select the Model and Transverse depth and click the 'Make ERD(html)' button
