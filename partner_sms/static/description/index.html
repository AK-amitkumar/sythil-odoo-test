Description

Allows you send SMS's using the smsglobal gateway
This module only sends smses, it does not yet recieve them

Using the module

Initial Configuration

1. Create a smsglobal account and add credit
2. Go to Settings->SMS Gateway Configuration->Accounts and enter the all the api settings
3. Add widget="smsfield" to any field you wish treat as a mobile number

Sending SMS

1. Go to the Partner/Customer you wish to sms
2. Make sure the mobile number is in the international standard e.g. 610437999111, + and spaces are acceptable
3. Click on the underlined mobile number
4. Select the account/number you wish to send from
5. Write the sms and hit "Send SMS"

View SMS Histroy

1. Go to Settings->SMS Gateway Configuration->SMS History
