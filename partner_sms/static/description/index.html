Description

Allows you send SMS's to Partners
Uses smsglobal as the sms gateway
This module only sends smses, it does not yet recieve them

Using the module

1. Create a smsglobal account and add credit
2. Go to Settings->SMS Gateway Configuration->Accounts and enter the all the api settings
3. Go to the Partner/Customer you wish to sms
4. Make sure the mobile number is in the international standard e.g. 610437999111, + and spaces are acceptable
5. Hit the "Compose SMS"
6. Select the account/number you wish to send from
7. Write the sms and hit "Send SMS"
8. View the sent smses via Settings->SMS Gateway Configuration->SMS History
